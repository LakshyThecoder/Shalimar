# Shalimar

Ristorante Indiano Halal — Milano

Cinematic multilingual website for Shalimar (Viale Toscana 2, Milan).

## Run

```bash
npm install
npm run dev
```

## Routes

- `/` — Home
- `/menu` — Display menu (no prices)
- `/qrmenu` — QR menu (with prices)

## Stack

React · Vite · Tailwind · Framer Motion · Lenis · react-i18next
